# Zotero 文库架构模板

> 适用场景：论文文献管理
> 参考大纲：研究方向（由 Step 1-2 确定）

## 集合结构

```
📁 论文文献库 （根集合）
│
├── 📁 1-基础
│   ├── P0-A1 理论基础
│   │   基础理论
│   ├── P0-A2 综述
│   │   领域综述论文
│   └── P0-A3 方法综述
│       方法综述
│
├── 📁 2-方向一
│   ├── P1-D1 子方向A
│   │   描述
│   ├── P1-D2 子方向B
│   │   描述
│   └── P1-D3 子方向C
│       描述
│
├── 📁 3-方向二
│   ├── P2-E1 子方向A
│   │   描述
│   └── P2-E2 子方向B
│       描述
│
└── 📁 4-方向三
    ├── P3-F1 子方向A
    │   描述
    └── P3-F2 子方向B
        描述
```

## 标签方案

| 标签 | 所属章节 | 说明 |
|------|----------|------|
| P0-A1 | 基础-理论 | 基础理论 |
| P0-A2 | 基础-综述 | 综述 |
| P1-D1 | 方向一-子A | 描述 |
| P1-D2 | 方向一-子B | 描述 |
| P2-E1 | 方向二-子A | 描述 |
| P3-F1 | 方向三-子A | 描述 |

## 使用说明

1. 在 Zotero 桌面端 → 文件 → 新建文库
2. 按上述结构新建集合（右键 → 新建集合）
3. 导入论文后，选中论文 → 拖入对应集合
4. 批量选择 → 右键 → 添加标签 → 输入对应标签
