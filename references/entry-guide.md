# Entry Guide

This guide helps readers choose the right entry point for the more-paper workflow without replaying the whole pipeline.

| Current need | Entry point | What must be known first | Standard output |
|---|---|---|---|
| Only have a research direction | Step 1 | Target topic, constraints, and current status | `研究主题.md` |
| Have a topic and need a paper plan | Step 2 | Existing outline or chapter intent | `大纲关键词.md` |
| Need a search plan | Step 3 | Scope, source preference, and evidence needs | `检索方案.md` |
| Need search and scoring | Step 4 | Query set and screening rules | `检索文献表.md`, `workflow_search_results.json` |
| Need PDF download | Step 5 | DOI, title, URL, BibTeX, or search results | PDF pool, download log, manifest |
| Need Zotero organization | Step 6 | BibTeX, PDF pool, and target library structure | Zotero plan and mapping artifacts |
| Need writing or section drafting | Step 7 | Evidence base and target chapter | Draft, evidence matrix, audit outputs |
| Need conservative polishing | Step 8 | Existing draft and revision scope | Polished draft and revision ledger |

## Quick routing rules

- Use Step 5 when the task is about downloading or resolving access.
- Use Step 6 only when the task is about library organization or attachment alignment.
- Use Step 7 when the task is about producing or extending正文草稿.
- Use Step 8 when the task is about tightening an existing draft without adding new evidence.

## Do not use this guide as a workflow lock

Any step may still be entered conversationally if the user already has equivalent input for that step. Artifact Passport based artifact readiness starts at Step 4 and covers Step 4-8 only.
