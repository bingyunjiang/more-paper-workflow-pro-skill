# 术语别名映射表 (Terminology Alias Mapping)

> 跨步骤术语标准化参考。确保 Step 2（关键词）、Step 3（检索查询）、
> Step 7（写作）和 Step 8（润色）使用的术语保持一致。
>
> 设计原则：**同一概念在全文中的术语形式不超过 1 个。**
> 借鉴来源：ResearchWiki `memory/term_aliases.md`。
>
> ⚠️ 本文件为模板。运行时由 agent 自动复制到项目目录下的 `.skill-state/term_aliases.md`。
> 修改请到 `.skill-state/` 中进行，以免污染技能仓库。

## 映射格式

每行一个术语：

| 字段 | 说明 |
|------|------|
| **Main Term** | 推荐使用的标准术语（中文论文用中文形式，英文论文用英文形式） |
| **Aliases** | 别名、同义词、缩写（用 `; ` 分隔） |
| **Recommended page** | 该术语主要出现的大纲章节（如 Ch2 引言、Ch3 方法） |
| **Note** | 使用注意事项（上下文差异、避免混淆的点、中文特有表述） |

## 使用规则

### 1. Step 2 执行时（关键词 → 术语映射）
- 从 `大纲关键词.md` 中提取**所有中英文术语对**
- 为每个术语确定 Main Term（推荐形式）、Aliases（同义词/缩写）、Recommended page（对应章节）
- 填充上表，覆盖空行或追加新行
- 同义词展开规则：英文同义词从检索经验中获取；中文同义词从领域习惯中获取

### 2. Step 3 执行前（检索查询 → 术语对齐）
- 读取本表，确保检索查询中的核心关键词与 Main Term 一致
- 将 Aliases 列的内容作为 OR 查询的候选词
- 发现新的术语变体时追加到 Aliases 列

### 3. Step 7 写作时（正文 → 术语对齐）
- 每章写作前，确认本章核心术语与本表中 Recommended page 匹配
- 中文论文：全篇统一使用 Main Term 的中文形式
- 英文论文：全篇统一使用 Main Term 的英文形式

### 4. Step 8 润色时（统一校验 → 术语一致性）
- 以本表为基准，逐章扫描正文，提取所有核心术语
- 对比映射表：术语是否与 Main Term 一致？是否存在同义词轮换？
- 不一致处理：
  - 修正为 Main Term → 记录到 before/after 表
  - 如果 Main Term 选择不当 → 更新本表 → 记录到 `.skill-state/decision_log.md`
  - 如果属于新术语 → 追加到本表

### 5. 本表是活文档
- 随着研究深入，术语选择可能变化
- 每次变更记录到下方的变更记录表
- 重大术语变更（影响 >3 章）记录到 `.skill-state/decision_log.md`
