# 章节-功能矩阵

用于 Step 2 和 Step 7，把“章节标题”转换成“章节功能 + 证据需求 + 写作责任”。

## 推荐字段

| 字段 | 说明 |
|---|---|
| section_id | 章节编号 |
| section_title | 章节标题 |
| section_function | 本章在整篇中的作用 |
| key_claim | 本章核心论点 |
| evidence_needed | 需要什么类型的文献或数据 |
| keyword_groups | 关联关键词组 |
| expected_output | 预期生成什么文本/图表/表格 |

## 常见 `section_function`

- `problem-framing`
- `literature-synthesis`
- `theoretical-foundation`
- `method-definition`
- `experiment-or-case`
- `comparison-and-discussion`
- `conclusion-and-outlook`

## 使用边界

- 不要求所有章节都同等详细
- 重点章节必须写清 `key_claim + evidence_needed`
