# 中文下载清单

## 最低必需字段

CNKI / 万方下载最小契约：

| 字段 | 必需 | 说明 |
|------|:--:|------|
| `title` | 是 | 仅用于日志、人工核对 |
| `source` | 是 | 只能是 `cnki` 或 `wanfang` |
| `article_url` | 是 | 真正驱动下载的详情页 URL |

## 推荐补齐字段

这些字段不是脚本最低门槛，但强烈建议一起带上：

| 字段 | 作用 |
|------|------|
| `source_id` | 便于稳定标识和后续回查 |
| `doi` | 若存在，可用于去重和回溯 |
| `authors` | 结果核对 |
| `year` | 结果核对 |
| `publication_title` | 期刊/学位信息核对 |
| `abstract` | 人工确认是否命中目标文献 |
| `citekey` | 与 BibTeX 对齐 |
| `zotero_item_key` | 回查 Zotero |
| `status` | `ready / unresolved` 管理 |
