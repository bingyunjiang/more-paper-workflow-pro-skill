## 观察点

- Windows 下 `utf-8` / `cp936` 控制台输出是否乱码
- `BibTeX` 读入后 DOI 解析是否正常
- 日志里 DOI 前缀分流是否正确
- `ScienceDirect` 会话与超时是否稳定
- 中文题名、作者、文件名是否导致异常
