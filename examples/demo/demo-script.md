# Demo Script

这个 demo 用来复现“第一次成功”的最短路径。第一轮不要求录屏，先用文本回放版固定输入、关键反馈和最终产物。

## Demo 目标

- 入口：Step 5 直达下载
- 目的：证明这个 skill 不要求线性从 Step 1 走到 Step 5
- 输出：`download_manifest.json` 风格的下载路由摘要

## 固定输入

```text
请直接进入 Step 5。根据这 3 个 DOI 生成下载清单并说明下载路由，不要回跑前面的检索步骤：10.1016/j.est.2024.113105; 10.1109/ACCESS.2024.3399912; 10.1016/j.energy.2022.125097
```

## 预期关键反馈

1. 明确识别为 Step 5 直达入口
2. 明确说明不会强制回跑 Step 1-4
3. 生成下载清单并给出路由预览
4. 如有登录/权限风险，显式提示而不是静默跳过

## 预期产物

- `download_manifest.json`
- `unresolved_download_items.md`（若出现失败或待人工处理）
