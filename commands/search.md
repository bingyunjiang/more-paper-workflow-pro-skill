# Search

## 何时使用
- 已有研究主题或章节目录，需要做 Step 3/4 的检索方案与检索执行
- 想得到 `文献库.bib`、`检索文献表.md`、`检索报告.md`
- 需要多源检索、评分分级、引文扩展或饱和度分析

## 最小输入
- `研究主题.md` 或 `大纲关键词.md`
- 可选：已有 BibTeX、JSON 检索结果、中文元数据、指定数据库偏好

## 主要产出
- `检索方案.md`
- `检索文献表.md/.xlsx`
- `检索报告.md/.pdf`
- `文献库.bib`

## 常见阻塞点
- 主题未收敛，检索式过宽
- 章节证据需求不清，导致评分与写作脱节
- 中文文献需要保留 `source_id` / `article_url`，不能只靠 DOI
