# 热变形过程中动态再结晶驱动晶粒细化的多尺度机理

热变形条件下的晶粒细化并不是单一热激活过程，而是应力状态、位错储能、亚晶结构演化和晶界迁移共同作用的结果。对于铝合金热加工，连续动态再结晶（CDRX）通常占据主导地位，其核心路径可以概括为：塑性变形首先引起位错累积和动态回复，位错胞壁进一步转化为低角度亚晶界；随着应变持续增加，亚晶界取向差逐步增大，部分低角度晶界向高角度晶界转化，最终形成新的细化晶粒。Liu 等在 AA2196 合金中将 VPSC 与三维 CA 耦合，明确把应力响应、位错累积/回复、晶体取向演化、亚晶形成与旋转以及储能/曲率驱动的边界迁移放入同一模拟框架，说明 CDRX 的机理不能仅用平均晶粒尺寸或宏观流变曲线解释，而应追踪“滑移激活 -> 位错储能 -> 亚晶旋转 -> 高角度晶界形成”的连续链条（Liu et al., 2024（已读全文））。

应力状态进一步改变上述链条中亚晶和晶界的空间组织方式。Yu 等针对 AA2195 设计了平面应变压缩（PSC）、单轴压缩（UC）和扭转（TR）三类加载路径，结果表明，压缩主导状态下晶粒细化机制由晶粒破碎（GF）向亚晶旋转（GR）转变的温度约为 350 C，而剪切主导的扭转状态下两种机制在约 300 C 即可共同激活（Yu et al., 2025（已读全文））。这说明机制转换阈值不是固定材料常数，而是由加载路径诱导的前驱亚结构决定。压缩状态更容易使轧制态片层晶粒继续压扁，并在晶内形成取向较随机的低角度晶界，因而更偏向晶粒破碎；剪切状态则促使低角度晶界沿初始晶粒短轴优先生长，并更容易延伸至三叉晶界，从而降低亚晶旋转和新晶粒团簇形成的结构门槛。由此看，热变形中的晶粒细化机理应以“应力状态 -> 前驱亚结构 -> 晶界连通方式 -> GF/GR 主导关系”作为主线，而不是简单归因于温度或应变速率升降。

宏观流动应力的硬化或软化，是上述微观机制竞争后的外在表现。加工硬化（WH）来源于变形过程中的位错增殖，动态回复（DRV）通过位错湮灭与重排削弱硬化，动态再结晶（DRX）则通过新晶粒形成和晶界迁移进一步释放储能。Yu 等的结果显示，在相同总应变下，DRV 对软化的贡献在 PSC 中最明显，而 DRX 诱导软化在 TR 中最显著（Yu et al., 2025（已读全文））。这一差异可由激活能和再结晶体积分数共同解释：压缩主导条件下较强的回复过程可较早抵消位错硬化，但剪切主导条件更容易形成促进亚晶旋转和三叉晶界连接的局部结构，使 DRX 更充分地介入软化过程。因此，流动曲线中的峰值、稳态或持续软化不应被直接视作某一种机制的独立证据，而应回到位错密度、低/中/高角度晶界比例、再结晶分数和亚晶取向梯度进行交叉判断。

从数值建模角度看，机理研究的难点在于把连续力学场与离散组织演化统一起来。传统 CA 适合描述形核、晶界迁移和晶粒拓扑变化，但离散状态跳变容易与 CPFE 中连续应力/应变场不兼容；Chen 等提出用连续转变分数描述有限元子网格内从变形态到再结晶态的渐进重排，并通过双向映射在每一时间步交换位错密度、晶体取向和转变状态，从而缓解 CA-CPFE 耦合中的非物理应力突跳（Chen et al., 2026（已读全文））。这一思路对 CDRX 和 DDRX 的共同启示是：再结晶不能被处理为“某一单元瞬间变成新晶粒”的黑箱事件，而应描述为储能释放、取向更新、边界迁移和局部软化反馈逐步展开的过程。尤其在 DDRX 中，形核和晶界迁移会形成局部软区，先释放弹性畸变能并造成局部应力下降，随后再通过边界迁移消耗变形基体和位错密度（Chen et al., 2026（已读全文））。这一反馈链条为解释“微观组织演化如何反过来改变宏观流动应力”提供了可计算路径。

综合来看，动态再结晶驱动晶粒细化的机理可归纳为三个层级。第一层是位错和亚晶层级：位错累积、回复、低角度晶界形成和取向差增长决定了亚晶能否向新晶粒演化。第二层是加载路径层级：不同应力状态通过改变低角度晶界取向、三叉晶界连通和局部剪切带形成，改变 GF 与 GR 的竞争关系。第三层是多尺度耦合层级：CPFE/VPSC 提供晶粒尺度的非均匀变形和取向演化，CA 或连续场 CA 描述形核、转变分数与晶界迁移，两者共同连接宏观流动应力与微观组织重排。因而，在后续机理研究中，应优先建立“状态量-作用路径-证据锚点”的分析框架：以应力状态、温度、应变速率、位错密度、取向差和转变分数为关键状态量，以亚晶旋转、晶粒破碎、形核和晶界迁移为作用路径，并用 EBSD/TKD/TEM、流变曲线和多尺度模拟结果共同验证。只有当这些证据链同时闭合时，才能将观察到的晶粒细化现象提升为可靠的机理判断。
